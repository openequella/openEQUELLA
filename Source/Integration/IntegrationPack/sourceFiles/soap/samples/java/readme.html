Building

Some third party library jars are required to be downloaded before the project can be compiled and executed. Download the following jars and place in the ./lib directory:

  • axis-1.4.jar
  • commons-codec-1.9.jar
  • commons-discovery-0.5.jar
  • commons-logging-1.1.3.jar
  • jaxrpc-api-1.1.jar
  • wsdl4j-1.6.3.jar

To build the example jar you will need to have Apache Ant installed. Once installed, make sure the ant executable is on your PATH, and run the following command from the directory containing this readme.html file:

ant product

Executing

To execute the example project, change to the "product" directory and run the following:

java -classpath "soapclient.jar;*" org.example.soap.Main

This will perform a search using the query "test" and contribute an image attachment to the collection "Basic Items". You can edit these defaults in the settings.properties file within src/org/example/soap and then rebuilding the project. Ideally these values would be supplied by an end-user, but this project exists as an example only.
